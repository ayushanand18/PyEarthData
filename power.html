POWER Module¶

Fetching NASA/POWER CERES/MERRA2 Native Resolution Daily Data.

pyearthdata.power.get_temp(longitude, latitude, start, end, **kwargs)[source]¶

Fetch temperature data at a specific date and (lat, lon) pair

Parameters:

  • longitude – [Float] Longitude e.g. 77.0000

  • latitude – [Float] Latitude

  • start – [Integer] Start date. format YYYYMMDD

  • end – [Integer] End Date. format YYYYMMDD

Returns:

An Image

Usage:

# from PyEarthData import power
res = power.get_temp(77,22,20141201,20141205)
print(res)
print(res["geometry"])
